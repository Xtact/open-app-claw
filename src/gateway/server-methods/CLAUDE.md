# Project Context

## Commands
- **Serve (Local):** `bundle exec jekyll serve`
- **Build:** `bundle exec jekyll build`
- **Install Dependencies:** `bundle install`
- **Clean:** `bundle exec jekyll clean`

## Tech Stack
- Static Site Generator: Jekyll
- Hosting: GitHub Pages
- Language: Ruby, Markdown, HTML, SCSS

## Code Style Guidelines
- **Markdown:** Use standard Markdown for content.
- **Front Matter:** Ensure all `.md` and `.html` pages start with valid YAML front matter.
- **Assets:** Place images in `/assets/images/`.
- **Layouts:** Use `_layouts` for page templates and `_includes` for reusable components.
